NODO | Prototipo web frontend

Incluye:
- Login con usuario, contraseña y rol.
- Pantalla de registro para crear cuenta.
- Roles: Estudiante, Egresado y Contratante.
- Estudiante: acceso gratuito.
- Egresado: selección de suscripción de $20 MXN mensual o $200 MXN anual.
- Contratante: prueba gratuita de 7 días + plan mensual o anual.
- Inicio, explorar, ofertas, mensajería, notificaciones, perfil y plan.
- Filtros por skills de diseño multimedia.
- Perfil editable con skills y biografía.
- Creación de proyectos de portafolio.
- Subida de archivos desde la computadora para proyectos.
- Publicación de ofertas de trabajo.
- Propuestas con precio a tratar.
- Mensajería entre talento creativo y contratantes.
- Notificaciones dentro del sistema.
- Guardado local con localStorage.

Cómo usar:
1. Abre index.html en tu navegador.
2. Elige Iniciar sesión o Crear cuenta.
3. En registro, completa nombre, usuario, correo, contraseña, rol y plan si aplica.
4. En login, escribe usuario, contraseña y selecciona rol.
5. Si eres estudiante o egresado, puedes editar perfil, skills y subir proyectos.
6. Si eres contratante, puedes publicar ofertas y contactar talento.

Notas:
- Este proyecto funciona en frontend, sin base de datos.
- La información se guarda en tu navegador con localStorage.
- Para una versión real se puede conectar con PHP + MySQL.
